Adds Clubhouse Tags to the blog feed automatically.

== Description ==

This plugin adds Microsoft Clubhouse tags to your blog feed so your posts can
propagate on the Clubhouse website and on affiliated websites.

Find out more about the Microsoft Clubhouse at [http://clubhouse.microsoft.com/](http://clubhouse.microsoft.com/).

== Installation ==

1. Upload the plugin to your plugins folder: 'wp-content/plugins/'.
2. Activate the 'Clubhouse Tags' plugin from the Plugins admin panel.
3. Test this out by checking the source code of your blog feed. If you are using
   Feedburner, you might have to wait until Google refreshes your feed.

== Frequently Asked Questions ==

= What is the Microsoft Clubhouse? =
The Microsoft Clubhouse is a website collecting blog posts about Microsoft,
Windows and Windows Live-related products and services.

= How does the Clubhouse get my posts? =
You have to specify your blog feed URL upon sign-up.

= How can I become part of the Clubhouse? =
Find out more about the Microsoft Clubhouse at [http://clubhouse.microsoft.com/](http://clubhouse.microsoft.com/).

= I signed up! Where are my posts? =
If you want your posts to appear in the Microsoft Clubhouse, you should use
special tags so the Clubhouse feed spider can recognize your posts. This plugin
adds the necessary tags to your blog feed automatically.

= Great! What must I do now? =
Write a new post, and add the tag "Clubhouse" to it. Then add more tags based
on your post's content, like "Messenger" or "Writer", if you are talking about
them in your post.

= I can't get it to work! I'm frustrated! =
You can always catch me on [Twitter](http://twitter.com/mynetx).
You might also want to ask the [Clubhouse community manager](http://twitter.com/mattbernardy) for help.
